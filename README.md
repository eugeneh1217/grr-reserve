# Description
This bot shortens the booking process for the Grand River Rocks booking online booking system implimented during the COVID-19 Pandemic. It brings user to the last page of booking.  
Last updated 3/17/2021.  
Written in Python 3.8.5

# Usage
Install required modules:
```
> pip install -r requirements.txt
```

Run bot:
```
> python -m bot
```
